What is real here: everything you can see move. K parallel Gibbs chains run continuously in this page and are never restarted; pins and prompt edits are clamps applied to live state. Glow = measured flip rate, blur = measured slot entropy, ghosts = the ensemble’s actual recent candidates (EMA), the published line = the lowest-energy configuration actually visited, the energy trace = the real mean NLL in nats (lower is better). β=1 samples the model’s true conditional distribution; annealing is a real β schedule. Backward information flow is real: a clamped ending propagates leftward through the chain factors.

What is a stand-in: the energy model. A browser cannot run LLaMA, so this page uses a bigram chain-MRF built from the 126-sentence original corpus below (view source) — not a transformer. The runtime semantics (energy + clamp mask + schedule, observe→constrain→relax→publish) are identical to EXP-011 on THRML; the transformer-energy version of this exact interface is EXP-012 on the Lightning L4, with the mechanism receipt already on file (EXP-010: clamped-Gibbs infilling through GPT-2 attention, suffix-NLL +8.77/+11.50/+2.64 nats vs forward-only). This demo runs exact heat-bath Gibbs, so there is no accept-rate gauge — accept/reject is a Metropolis quantity and will appear on the LLaMA build, where it is real.
